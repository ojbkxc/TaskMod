TTS 语音合成 API

通过 Android 标准的 TextToSpeech 子系统控制设备 TTS。支持 Google TTS、小爱同学、讯飞、百度等任意已安装引擎。

GET /api/tts/engines 获取设备上所有可用 TTS 引擎列表

通过 cmd tts list engines、pm list packages、dumpsys texttospeech 三种方式联合发现引擎，自动去重。默认引擎标记为 [默认] 并排在首位。

响应示例

// 200 OK
{
  "success": true,
  "message": "ok",
  "data": [
    { "package_name": "com.google.android.tts", "label": "[默认] Google 文字转语音" },
    { "package_name": "com.iflytek.tts", "label": "讯飞 TTS" },
    { "package_name": "com.miui.tts", "label": "小米 TTS" }
  ]
}

POST /api/tts/speak 朗读指定文本

请求体 (JSON)

字段类型必填说明
textstring是要朗读的文本内容
enginestring否TTS 引擎包名，如 com.google.android.tts
languagestring否语言代码，如 zh-CN、en-US
ratefloat否语速，0.5~2.0，默认 1.0
pitchfloat否音调，0.5~2.0，默认 1.0
volumefloat否音量，0.0~1.0，默认 1.0（通过系统设置预设）

请求示例

{
  "text": "你好，欢迎使用 TaskMod",
  "engine": "com.google.android.tts",
  "rate": 1.2,
  "pitch": 1.0
}

响应示例

// 成功
{ "success": true, "data": "语音播放成功", "message": "你好，欢迎使用 TaskMod" }
// 失败
{ "success": false, "data": null, "message": "TTS 播放失败，请确认设备已安装 TTS 引擎..." }

在线测试

POST /api/tts/stop 停止当前正在播放的 TTS

先尝试 cmd tts stop，失败则 force-stop 常见 TTS 引擎进程。

响应示例

{ "success": true, "message": "语音播放已停止" }
GET PUT /api/tts/settings 获取 / 更新 TTS 全局设置

GET 响应

{
  "success": true,
  "data": {
    "default_engine": "com.google.android.tts",
    "default_rate": 1.0,
    "default_pitch": 1.0,
    "default_volume": 1.0,
    "available_engines": [ ... ]
  }
}

PUT 请求体

字段类型说明
ratefloat语速 0.5~2.0
pitchfloat音调 0.5~2.0
volumefloat音量 0.0~1.0
POST /api/tts/default-engine 设置系统默认 TTS 引擎

请求体

{ "engine": "com.google.android.tts" }

响应

{ "success": true, "message": "已设置默认TTS引擎为: com.google.android.tts" }

在线测试

POST /api/tts/test 快速测试 TTS 功能

使用默认参数播放测试语音，适合验证 TTS 是否正常工作。

请求体 (均可选)

{
  "text": "这是一段测试语音",  // 可选，默认为内置测试文本
  "engine": "com.google.android.tts"  // 可选，默认使用系统默认引擎
}

cURL 调用示例

# 获取所有 TTS 引擎
curl http://localhost:2718/api/tts/engines

# 朗读文本（使用默认引擎）
curl -X POST http://localhost:2718/api/tts/speak \
  -H "Content-Type: application/json" \
  -d '{"text":"你好世界","engine":"com.google.android.tts","rate":1.2}'

# 停止朗读
curl -X POST http://localhost:2718/api/tts/stop

# 获取 TTS 设置
curl http://localhost:2718/api/tts/settings

# 更新语速和音调
curl -X PUT http://localhost:2718/api/tts/settings \
  -H "Content-Type: application/json" \
  -d '{"rate":1.5,"pitch":1.2}'

# 设置默认引擎
curl -X POST http://localhost:2718/api/tts/default-engine \
  -H "Content-Type: application/json" \
  -d '{"engine":"com.google.android.tts"}'

# 快速测试
curl -X POST http://localhost:2718/api/tts/test \
  -H "Content-Type: application/json" \
  -d '{}'
