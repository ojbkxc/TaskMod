MQTT使用说明

发布设备状态: {prefix}/status - 设备电量、温度、运行状态等JSON数据（每分钟更新）

接收远程命令: {prefix}/cmd - 订阅此topic接收ADB命令

命令执行结果: {prefix}/result - 命令执行后的输出结果

示例: mosquitto_pub -t "taskmod/cmd" -m "input keyevent 3"

邮件通知说明

自动通知: 启用后，脚本执行完成会自动发送邮件通知

重试机制: 网络错误时会自动重试，最多重试配置的次数

附件支持: 支持发送带附件的邮件（工作流节点配置）

变量模板: 邮件主题和内容支持 {script} {time} {date} {result} 变量
